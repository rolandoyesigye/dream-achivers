## 🔧 Customization Tips

### To Change Colors

Edit `tailwind.config.js` and modify the color values in the `theme.extend.colors` section.

### To Add More Projects

Edit `index.html` and duplicate the project card structure in the Projects section.

### To Modify Services

Edit the service cards in the Services section of `index.html`.

### To Update Images

Replace images in the `/public` folder with your own (keep the same names or update references in HTML).
